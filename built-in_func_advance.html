1. 什麼是內建函式 (Built-in Functions)？

內建函式 (Built-in Functions) 是 Python 語言預先定義好的一組函式，您無需進行任何導入 (import) 操作，就可以在任何 Python 程式碼中直接使用它們。它們提供了許多常用且基礎的功能。

本頁將介紹 Python 中較為進階的內建函式，這些函式通常涉及更抽象的程式設計概念（如迭代器、高階函式、物件內省等），或者在特定情境下才需要使用。

2. 迭代器與序列函式 (Iterator and Sequence Functions)

2.1 `enumerate()`：同時獲取索引和值

在遍歷可迭代物件 (iterable) 時，同時提供元素的索引 (index) 和值 (value)。 

fruits = ["apple", "banana", "cherry"]
for index, fruit in enumerate(fruits):
    print(f"索引 {index}: {fruit}")
# 輸出:
# 索引 0: apple
# 索引 1: banana
# 索引 2: cherry

2.2 `zip()`：打包多個迭代器

將多個可迭代物件 (iterable) 中對應位置的元素打包成元組 (Tuple) 的序列。 

names = ["Alice", "Bob"]
ages = [25, 30]
cities = ["New York", "London"]

for name, age, city in zip(names, ages, cities):
    print(f"{name} ({age} 歲) 住在 {city}")
# 輸出:
# Alice (25 歲) 住在 New York
# Bob (30 歲) 住在 London

2.3 `map()`：對可迭代物件的每個元素應用函式

map(function, iterable, ...) 函式會將指定的函式 (function) 應用於可迭代物件 (iterable) 的每個元素，並返回一個 map 物件（這是一個迭代器 (iterator)）。您通常需要將其轉換為清單 (List) 或元組 (Tuple) 來查看結果。 

# 範例 1: 將清單 (List) 中的數字都平方
numbers = [1, 2, 3, 4]
squared_numbers = list(map(lambda x: x**2, numbers))
print(squared_numbers) # 輸出: [1, 4, 9, 16]

# 範例 2: 將字串 (String) 清單 (List) 中的每個字串 (String) 轉換為大寫
words = ["apple", "banana", "cherry"]
upper_words = list(map(str.upper, words))
print(upper_words) # 輸出: ['APPLE', 'BANANA', 'CHERRY']

# 範例 3: 結合多個可迭代物件 (iterable)
num1 = [1, 2, 3]
num2 = [4, 5, 6]
sum_of_lists = list(map(lambda x, y: x + y, num1, num2))
print(sum_of_lists) # 輸出: [5, 7, 9]

3. 實用工具函式 (Utility Functions)

3.1 `dir()`：列出物件的屬性與方法

返回指定物件的所有有效屬性 (attributes) 和方法 (methods) 的清單 (List)。如果沒有參數，則返回當前作用域 (scope) 中的名稱清單 (List)。 

my_string = "hello"
print(dir(my_string)) # 輸出: ['__add__', ..., 'upper', 'strip', ...] (部分輸出)

# print(dir()) # 輸出當前作用域 (scope) 中的所有名稱

3.2 `help()`：獲取幫助資訊

用於獲取函式、模組 (Module)、類別 (Class) 或關鍵字 (keyword) 的幫助資訊。 

# help(print) # 會顯示 print 函式的詳細說明

# help(len) # 會顯示 len 函式的詳細說明

3.3 `id()`：取得物件的唯一識別符

返回物件的唯一識別符 (identity)，在物件的生命週期中是唯一的。 

a = 10
b = 10
c = [1, 2]
d = [1, 2]

print(id(a)) # 輸出: 數字，例如 140735870000000 (a 和 b 通常指向同一個物件)
print(id(b)) # 輸出: 相同的數字

print(id(c)) # 輸出: 數字，例如 140735870000123
print(id(d)) # 輸出: 不同的數字 (清單 (List) 是可變的，即使內容相同也是不同物件)

3.4 `eval()`：執行字串 (String) 中的 Python 程式碼

將字串 (String) 作為 Python 表達式 (expression) 進行求值並返回結果。使用時需非常小心，因為它可能引入安全漏洞。 

expression = "2 + 3 * 4"
result = eval(expression)
print(result) # 輸出: 14

# 可以執行變數
x = 10
y = 5
code_str = "x * y"
print(eval(code_str)) # 輸出: 50
