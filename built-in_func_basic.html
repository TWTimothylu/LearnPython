1. 什麼是內建函式 (Built-in Functions)？

內建函式 (Built-in Functions) 是 Python 語言預先定義好的一組函式，您無需進行任何導入 (import) 操作，就可以在任何 Python 程式碼中直接使用它們。它們提供了許多常用且基礎的功能，是理解 Python 程式碼的基石。

2. 輸出/輸入與基本資訊 (Basic I/O & Information)

2.1 `print()`：輸出資訊

用於將資訊輸出到控制台 (console) 或標準輸出 (standard output)。 

print("Hello, Python!") # 輸出字串 (String)
print(123)             # 輸出數字
name = "Alice"
print("我的名字是", name) # 輸出多個項目，用空格分隔

# 使用 f-string 格式化輸出
age = 30
print(f"我今年 {age} 歲。")

2.2 `input()`：從使用者獲取輸入

用於從使用者那裡獲取輸入。所有輸入都會被讀取為字串 (String)。 

# user_name = input("請輸入您的名字: ")
# print(f"您好，{user_name}！")

# 注意：input() 總是回傳字串 (String)，如果需要數字，需要進行型態轉換 (Type Conversion)
# num_str = input("請輸入一個數字: ")
# num_int = int(num_str) # 將字串 (String) 轉換為整數 (Integer)
# print(f"您輸入的數字是: {num_int}")

2.3 `len()`：取得長度

返回物件的長度 (Length) 或項目數量，適用於序列 (sequence)（如字串 (String)、清單 (List)、元組 (Tuple)）和集合 (collection)（如字典 (Dictionary)、集合 (Set)）。 

my_string = "Python"
print(len(my_string)) # 輸出: 6

my_list = [1, 2, 3, 4]
print(len(my_list))   # 輸出: 4

my_dict = {"a": 1, "b": 2}
print(len(my_dict))   # 輸出: 2 (鍵的數量)

2.4 `type()`：取得物件型態

返回物件的資料型態 (Data Type)。 

print(type("hello"))  # 輸出: <class 'str'>
print(type(123))      # 輸出: <class 'int'>
print(type([1, 2]))   # 輸出: <class 'list'>

3. 基礎型態轉換 (Basic Type Conversion)

這些函式用於將一個資料型態 (Data Type) 的值轉換為另一個資料型態 (Data Type)。

3.1 `int()`：轉換為整數 (Integer)

print(int("123"))     # 輸出: 123
print(int(3.14))      # 輸出: 3 (截斷小數部分)
print(int(True))      # 輸出: 1

3.2 `float()`：轉換為浮點數 (Float)

print(float("3.14"))  # 輸出: 3.14
print(float(10))      # 輸出: 10.0
print(float(False))   # 輸出: 0.0

3.3 `str()`：轉換為字串 (String)

print(str(123))       # 輸出: "123"
print(str(3.14))      # 輸出: "3.14"
print(str([1, 2]))    # 輸出: "[1, 2]"

3.4 `bool()`：轉換為布林值 (Boolean)

大多數值在布林上下文中都被視為 `True`。以下值會被視為 `False`：

  • `None`
  • `False`
  • 數字 `0` (整數 (Integer)、浮點數 (Float)、複數 (Complex))
  • 空序列 (empty sequence)（`''`, `[]`, `()`）
  • 空映射 (empty mapping)（`{}`）
  • 空集合 (empty set)（`set()`）
print(bool(1))        # 輸出: True
print(bool(0))        # 輸出: False
print(bool("hello"))  # 輸出: True
print(bool(""))        # 輸出: False
print(bool([]))       # 輸出: False
print(bool(None))     # 輸出: False

4. 基礎數學函式 (Basic Mathematical Functions)

4.1 `abs()`：絕對值

返回數字的絕對值。 

print(abs(-10))   # 輸出: 10
print(abs(3.14))  # 輸出: 3.14

4.2 `min()`：最小值

返回可迭代物件 (iterable) 中的最小值，或者多個引數中的最小值。 

print(min(1, 5, 2))      # 輸出: 1
print(min([10, 2, 8]))   # 輸出: 2
print(min("apple", "banana")) # 輸出: apple (按字母順序)

4.3 `max()`：最大值

返回可迭代物件 (iterable) 中的最大值，或者多個引數中的最大值。 

print(max(1, 5, 2))      # 輸出: 5
print(max([10, 2, 8]))   # 輸出: 10
print(max("apple", "banana")) # 輸出: banana

4.4 `sum()`：總和

返回數字可迭代物件 (iterable) 中所有元素的總和。 

print(sum([1, 2, 3]))    # 輸出: 6
print(sum((10, 20, 30))) # 輸出: 60

4.5 `round()`：四捨五入

對數字進行四捨五入。可以指定保留的小數位數。 

print(round(3.14159))    # 輸出: 3
print(round(3.14159, 2)) # 輸出: 3.14
print(round(2.5))        # 輸出: 2 (注意：Python 3 遵循「四捨六入五成雙」規則)
print(round(3.5))        # 輸出: 4

5. 序列生成 (Sequence Generation)

5.1 `range()`：生成數字序列

用於生成一個數字序列，常用於 `for` 迴圈。

  • `range(stop)`：從 0 到 `stop-1`。
  • `range(start, stop)`：從 `start` 到 `stop-1`。
  • `range(start, stop, step)`：從 `start` 到 `stop-1`，每次跳過 `step`。
for i in range(3):
    print(i) # 輸出: 0, 1, 2

for i in range(1, 5):
    print(i) # 輸出: 1, 2, 3, 4

for i in range(0, 10, 2):
    print(i) # 輸出: 0, 2, 4, 6, 8

6. 排序 (Sorting)

6.1 `sorted()`：排序可迭代物件

返回一個新的排序清單 (List)，不修改原始可迭代物件 (iterable)。 

numbers = [3, 1, 4, 1, 5]
sorted_numbers = sorted(numbers)
print(sorted_numbers) # 輸出: [1, 1, 3, 4, 5]
print(numbers)        # 輸出: [3, 1, 4, 1, 5] (原始清單 (List) 不變)

# 降序排序
print(sorted(numbers, reverse=True)) # 輸出: [5, 4, 3, 1, 1]
