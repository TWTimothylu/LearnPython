1. 什麼是流程控制 (Flow Control)？

在程式設計中，**流程控制 (Flow Control)** 是指程式碼執行的順序和邏輯。它允許您的程式根據特定條件做出決策、重複執行某段程式碼，或者跳過某些部分，而不是簡單地從上到下直線執行。

流程控制語句是所有程式語言的基石，它們賦予了程式智慧和動態性。Python 提供了多種流程控制語句：

  • **條件判斷語句 (Conditional Statements)**：if, elif, else
  • **迴圈語句 (Looping Statements)**：for, while
  • **跳轉語句 (Jump Statements)**：break, continue, pass

2. 條件判斷語句 (Conditional Statements)

條件判斷語句允許您的程式碼根據條件的真假來執行不同的程式碼區塊。

2.1 `if` 語句

if 語句是最基本的條件判斷語句。如果條件為 True，則執行 if 區塊內的程式碼。 

score = 90
if score >= 60:
    print("及格了！") # 輸出: 及格了！

temperature = 20
if temperature > 25:
    print("天氣炎熱") # 這裡不會執行

2.2 `if-else` 語句

if-else 語句用於在條件為 True 或 False 時執行不同的程式碼區塊。 

num = 7
if num % 2 == 0:
    print("這是偶數")
else:
    print("這是奇數") # 輸出: 這是奇數

age = 17
if age >= 18:
    print("已成年")
else:
    print("未成年") # 輸出: 未成年

2.3 `if-elif-else` 語句

當您需要檢查多個條件時，可以使用 elif (else if 的縮寫)。程式碼會依序檢查條件，一旦有一個條件為 True，其對應的區塊就會執行，然後跳過其餘的 elif 和 else 區塊。 

grade = 85

if grade >= 90:
    print("等級：A")
elif grade >= 80:
    print("等級：B") # 輸出: 等級：B
elif grade >= 70:
    print("等級：C")
else:
    print("等級：D")

# 另一個例子
traffic_light = "yellow"
if traffic_light == "green":
    print("通行")
elif traffic_light == "yellow":
    print("準備停止") # 輸出: 準備停止
else:
    print("停止")

2.4 巢狀 `if` 語句 (Nested `if` Statements)

您可以在一個 if 或 elif 或 else 區塊內部再放置另一個 if 語句，形成巢狀結構。 

username = "admin"
password = "123"

if username == "admin":
    if password == "123":
        print("登入成功！") # 輸出: 登入成功！
    else:
        print("密碼錯誤。")
else:
    print("使用者名稱不存在。")

# 判斷是否為正偶數
num = 6
if num > 0:
    if num % 2 == 0:
        print("這是個正偶數。") # 輸出: 這是個正偶數。
    else:
        print("這是個正奇數。")
else:
    print("這不是個正數。")

3. 迴圈語句 (Looping Statements)

迴圈語句用於重複執行一段程式碼，直到滿足特定條件為止。

3.1 `for` 迴圈

for 迴圈用於遍歷 (iterate) 任何可迭代物件 (iterable)，例如列表 (List)、元組 (Tuple)、字串 (String) 或 range() 函數生成的序列。 

# 遍歷列表 (List)
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
# 輸出:
# apple
# banana
# cherry

# 遍歷字串 (String)
for char in "Python":
    print(char)
# 輸出:
# P
# y
# t
# h
# o
# n

# 使用 range() 函數遍歷數字序列
# range(stop): 從 0 到 stop-1
for i in range(5):
    print(i) # 輸出: 0, 1, 2, 3, 4

# range(start, stop): 從 start 到 stop-1
for i in range(1, 4):
    print(i) # 輸出: 1, 2, 3

# range(start, stop, step): 從 start 到 stop-1，每次跳過 step
for i in range(0, 10, 2):
    print(i) # 輸出: 0, 2, 4, 6, 8

# 計算列表 (List) 元素的總和
numbers = [10, 20, 30, 40]
total = 0
for num in numbers:
    total += num
print(f"總和: {total}") # 輸出: 總和: 100

3.2 `while` 迴圈

while 迴圈會只要條件為 True 就一直重複執行程式碼區塊。 

# 基本 while 迴圈
count = 0
while count < 3:
    print(f"計數: {count}")
    count += 1
# 輸出:
# 計數: 0
# 計數: 1
# 計數: 2

# 無限迴圈 (Infinite Loop) 和如何避免
# 下面的程式碼會一直執行，除非您手動停止它 (Ctrl+C)
# 注意：請不要在沒有停止條件的情況下執行 True: 的迴圈！
# while True:
#     print("我會永遠執行！")
#     break # 只有加上 break 才能停止

# 避免無限迴圈的關鍵是確保條件最終會變為 False
num_guesses = 0
correct_answer = 5
user_guess = 0

while user_guess != correct_answer:
    user_guess = int(input("猜一個數字 (1-10): ")) # 模擬使用者輸入
    num_guesses += 1
    if user_guess == correct_answer:
        print(f"恭喜！您猜對了，總共猜了 {num_guesses} 次。")
    else:
        print("猜錯了，請再試一次。")

4. `break` 和 `continue` 語句

這兩個語句用於在迴圈執行過程中改變其預設的流程。

4.1 `break` 語句

break 語句用於**立即終止**當前的迴圈 (for 或 while)，並跳到迴圈後的第一個語句。 

# 在 for 迴圈中使用 break
for i in range(10):
    if i == 5:
        print("找到 5 了，迴圈終止！")
        break # 遇到 5 就跳出迴圈
    print(f"當前數字: {i}")
# 輸出:
# 當前數字: 0
# 當前數字: 1
# 當前數字: 2
# 當前數字: 3
# 當前數字: 4
# 找到 5 了，迴圈終止！

# 在 while 迴圈中使用 break
count = 0
while True: # 無限迴圈
    print(f"計數: {count}")
    count += 1
    if count == 3:
        print("計數達到 3，退出迴圈。")
        break # 條件達成時退出
# 輸出:
# 計數: 0
# 計數: 1
# 計數: 2
# 計數達到 3，退出迴圈。

4.2 `continue` 語句

continue 語句用於**跳過**當前迴圈迭代中其餘的程式碼，並立即進入下一次迴圈迭代的判斷。 

# 在 for 迴圈中使用 continue
for i in range(5):
    if i == 2:
        print("跳過數字 2 的處理。")
        continue # 跳過 i=2 時，print(i) 的執行
    print(f"處理數字: {i}")
# 輸出:
# 處理數字: 0
# 處理數字: 1
# 跳過數字 2 的處理。
# 處理數字: 3
# 處理數字: 4

# 在 while 迴圈中使用 continue
count = 0
while count < 5:
    count += 1
    if count % 2 != 0: # 如果是奇數
        print(f"跳過奇數 {count} 的處理。")
        continue # 跳過奇數的後續程式碼
    print(f"處理偶數: {count}")
# 輸出:
# 跳過奇數 1 的處理。
# 處理偶數: 2
# 跳過奇數 3 的處理。
# 處理偶數: 4
# 跳過奇數 5 的處理。

5. `pass` 語句

pass 語句是一個空操作 (no-operation) 語句。當語法要求一個語句但您暫時不想寫任何程式碼時，可以使用它作為佔位符 (placeholder)。它不做任何事情。 

# 在條件判斷中使用 pass
age = 15
if age >= 18:
    pass # 這裡目前不做任何事
else:
    print("未成年人。") # 輸出: 未成年人。

# 在函式 (function) 中使用 pass (定義空函式)
def my_function():
    pass # 函式 (function) 暫時為空

my_function() # 執行不會有任何效果或錯誤

# 在迴圈中使用 pass
for i in range(5):
    if i % 2 == 0:
        pass # 偶數暫時不做處理
    else:
        print(f"奇數: {i}")
# 輸出:
# 奇數: 1
# 奇數: 3
